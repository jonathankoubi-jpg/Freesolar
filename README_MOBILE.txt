
FREESOLAR — ZIP unique spécial mobile

Ce dossier contient tout le site + .env.local déjà rempli.
Étapes rapides (sans ordi) :
1) Décompresse le ZIP dans l’app Fichiers.
2) Va sur github.com → crée un dépôt `freesolar` → Add file → Upload files.
3) Sélectionne TOUT le contenu de ce dossier → Commit.
4) Sur vercel.com/new → Import Git Repository → choisis `freesolar` → Deploy.
5) Si Vercel demande les variables d’env, copie/colle ce qu’il y a dans `.env.local`.
